发表于 2020-03-19 更新于 2020-03-26 分类于 肌电 ， 动态手势

这次的任务: 使用SSD框架进行手部的实时追踪.

环境的搭建

说明:笔记本配置是i5 7代,显卡GTX1066(支持CUDA),windows10.

tensorflow与CUDA/cuDNN版本对应表

要配置的环境(旧版)

由于object detection api的更新,我后来卡住了,只能装新版:

  1. anaconda3(64位);下个新的就好了.

  2. python3.6;anaconda里面设一个python3.6的环境就好了.

  3. CUDA-9.1; 下载地址https://developer.nvidia.com/cuda-toolkit-archive 安装完后下载下面的(会自动添加环境变量)

  4. CuDNN-7.0.5;下载地址https://developer.nvidia.com/rdp/cudnn-archive ，在下载CUDNN7.0.5之前，需要登录NVIDIA账号，没有的话要注册登录后才能下载. 注意添加环境变量

  5. TensorFlow-gpu(1.4.0版本):从网站:https://github.com/fo40225/tensorflow-windows-wheel 下载特定版本的whl文件,之后在conda环境里用(比如)pip install tensorflow_gpu-1.4.0-cp36-cp36m-win_amd64.whl

最终新环境(本来用TF2.1的,object detection api比较老,又换回TF1.15了):

  1. anaconda3(64位);下个新的就好了.

  2. python3.7;anaconda里面设一个python3.7的环境就好了.

  3. CUDA10.1+cuDNN7.6;

  4. tensorflow1.15;

  5. object detection api; 参考https://www.youtube.com/watch?v=qDm_YOwP2zY
    安装完后cd到models-master\research目录使用Python object_detection/builders/model_builder_test.py

需要学习的技能

  1. TensorFlow’s Object Detector API

对egohands数据集的介绍和工作

    首先egohands数据集里有48个不同场景的文件夹,每个文件夹有100张图片,每张图片的手部都进行了标注,每个图片分别有四个label,分别是 myleft,myright,yourleft,yourright.对应我和你的四只手是否有出现.对于出现的手进行了标注,根据 https://github.com/victordibia/handtracking 提供的工具,成功实现了(第一次用的四个,发生了点问题,先用单个试试)单个类别的mat to csv.接下来使用generate_tfrecord.py将csv转成tfrecord格式( 指导视频:https://pythonprogramming.net/creating-tfrecord-files-tensorflow-object-detection-api-tutorial/ )

训练模型前的准备

训练自己的模型or使用迁移学习拿别的数据集的成果进行参数训练,我选择后者,选择使用GPU训练,比cpu大概快10倍

  1. 我选择了ssd_mobilenet_v2_coco

  2. 根据要求修改ssd_mobilenet_v2_coco.config.

  3. 放入object_detection进行模型训练,WTF?又遇到了环境问题,因为使用的是tensorflow2.1,而目前object_detection api只支持到tensorflow1.15版本,因此运行时会遇到module 'tensorflow' has no attribute 'contrib'问题.又要切换tensorflow版本了天呐

  4. 于是我又卸载了TF2.1装了tensorflow1.15,还可以
